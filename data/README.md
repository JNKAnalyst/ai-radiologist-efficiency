# Data

This project is built on the **MIMIC-CXR** dataset of de-identified chest radiograph reports. MIMIC-CXR is gated by a PhysioNet data use agreement, so the raw data is **not** committed to this repository.

## Files

- `sample_reports.csv` — small synthetic sample (12 rows) with the same schema the pipeline expects. Provided so the notebook and KNIME workflow are runnable end-to-end without MIMIC-CXR access. The text is hand-written placeholder content, not real patient data.
- `mimic_cxr_reports.csv` — **not committed.** Place your locally downloaded MIMIC-CXR reports here (path is configured in `config/config.yaml`).
- `processed_output.csv` — **not committed.** Generated by running the NLP notebook or KNIME workflow.

## Schema

| Column | Type | Description |
|--------|------|-------------|
| `study_id` | int | MIMIC-CXR study identifier |
| `case_type` | str | `routine`, `complex`, or `review` |
| `ai_assisted` | int (0/1) | Whether the read used the AI assistant |
| `turnaround_minutes` | float | Minutes from study available to final read |
| `report_text` | str | Free-text radiology report |

## Obtaining MIMIC-CXR

1. Create a [PhysioNet](https://physionet.org/) account.
2. Complete CITI human-subjects training and submit the certificate.
3. Sign the MIMIC-CXR data use agreement.
4. Download the reports CSV and place it at the path defined in `config/config.yaml::data.mimic_cxr_path`.
